# 3X-UI-Rest-API-Telegram-Bot
# 🤖 ربات تلگرام مدیریت پنل 3X-UI — نسخه کامل

ربات تلگرامی کامل برای مدیریت پنل **3X-UI** به زبان فارسی.
شامل پنل ادمین، پنل مشتری، و سیستم فروش با پرداخت کارت‌به‌کارت.

---

## 🚀 راه‌اندازی سریع

```bash
pip install -r requirements.txt

export BOT_TOKEN="توکن_ربات_از_BotFather"
export CARD_NUMBER="6037991234567890"   # اختیاری — قابل تنظیم از ربات
export CARD_OWNER="علی محمدی"           # اختیاری

python bot.py
```

## 📋 امکانات

### 👑 پنل ادمین
| بخش | قابلیت‌ها |
|-----|----------|
| 📊 وضعیت سرور | CPU، RAM، آپتایم، ترافیک، وضعیت Xray |
| 🔗 اینباندها | لیست کامل + جزئیات + کلاینت‌ها |
| 👥 مدیریت کلاینت | افزودن، حذف، جستجو، ریست ترافیک |
| 🛒 مدیریت فروش | تعرفه‌ها، رسیدهای در انتظار، همه سفارش‌ها |
| 💳 تنظیم کارت | شماره کارت و نام صاحب کارت |

### 👤 پنل مشتری
| بخش | قابلیت‌ها |
|-----|----------|
| 📊 وضعیت اشتراک | حجم + نوار پیشرفت + روزهای مانده |
| 🛒 خرید / تمدید | انتخاب تعرفه، ارسال رسید |
| 📱 QR کد | کد اتصال برای اپ VPN |
| 📋 اطلاعات اتصال | IP، پورت، UUID، لینک sub |
| 🔔 وضعیت سفارش‌ها | پیگیری سفارش‌های ثبت‌شده |

### 🛒 جریان خرید
```
مشتری تعرفه انتخاب می‌کند
    ↓
شماره کارت + مبلغ نمایش داده می‌شود
    ↓
مشتری واریز کرده و تصویر رسید ارسال می‌کند
    ↓
ادمین نوتیف دریافت می‌کند (رسید + دکمه تأیید/رد)
    ↓
ادمین تأیید می‌کند
    ↓
سیستم هوشمند:
  • اگه کلاینت موجود باشد → تمدید (تاریخ انقضا اضافه می‌شود)
  • اگه نباشد → کلاینت جدید ساخته می‌شود
    ↓
مشتری نوتیف دریافت می‌کند + لینک اشتراک
```

## ⚙️ دستورات ربات

| دستور | توضیح |
|-------|-------|
| `/start` | انتخاب نقش (ادمین/مشتری) |
| `/setup` | تنظیم اتصال به پنل (ادمین) |
| `/setcard` | تنظیم شماره کارت واریز |
| `/reconnect` | اتصال مجدد به پنل |
| `/help` | راهنما |
| `/cancel` | لغو عملیات جاری |

---

## 💳 تنظیم کارت‌به‌کارت

دو روش:
1. متغیر محیطی: `CARD_NUMBER` و `CARD_OWNER`
2. دستور `/setcard` در ربات (یا از منوی مدیریت فروش)

فرمت ورود کارت:
```
6037991234567890
علی محمدی
```

---

## 🔐 محدود کردن دسترسی (اختیاری)

برای اینکه فقط ادمین‌های مشخص بتوانند `/setup` بزنند:
```python
ALLOWED_ADMINS = [123456789, 987654321]  # chat_id ها

# در ابتدای setup_start اضافه کنید:
if update.effective_user.id not in ALLOWED_ADMINS:
    await update.message.reply_text("⛔ دسترسی ندارید.")
    return ConversationHandler.END
```
